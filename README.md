# ani-auto-skip-extension 閉嘴動畫瘋

An extension to mute and wait for animation crazy's ADs
再也不用看動畫瘋聽到詐騙廣告。

## Installation

Load unpack development package > choose this directory
在開發者人員模式中載入未封裝的擴充功能

You can delete other files expect following:
可以只留以下幾個文件：

- main.js
- manifest.json
- service_worker.js
